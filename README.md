# 基于picker-view原生组件时间选择器

```bash
  此组件是基于小程序原生组件picker-view拓展的时间选择器，支持大多数类型的选择。
```
## 支持类型
- [x] year
- [x] year-month
- [x] date
- [x] datetime
- [x] time

## 类型year

```shell
 <picktime
  	title="默认选择周"
	type="year"
    value="{{ startDate }}"
	bind:getTimes="getTimesFn"
 ></picktime>
```
